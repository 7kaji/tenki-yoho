# tenki-yoho

## Requirement

- Bundler
- Yarn

## Dev

```
$ git clone git@github.com:7kaji/tenki-yoho.git
$ cd tenki-yoho
$ bundle install --path vendor/bundler
$ yarn install
$ bundle exec foreman start -f Procfile.dev

$ open http://localhost:8080
```
