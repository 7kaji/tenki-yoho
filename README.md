# tenki-yoho

## Requirement

- Ruby
- Bundler
- React
- Yarn

## Dev

```
$ git clone git@github.com:7kaji/tenki-yoho.git
$ cd tenki-yoho
$ bundle install --path vendor/bundler
$ bundle exec foreman start

$ yarn install
$ yarn run start
```
